# 구현체
<ul>
    <li>SPA 방식 적응을 위해 State로 페이지 분기.</li>
    <li>Ref를 활용한 답변 미완료 질문 스크롤 기능 구현.</li>
    <li>State로 정의한 Boolean 값을 통해 메뉴 Modal 구현.</li>
    <li>12^28 가지 경우의 수를 12가지 유형으로 구분하는 알고리즘 구현.</li>
    <li>사용자 편의성 및 주사용기기를 고려한 모바일 특화 디자인 UI 구현.</li>
</ul>
